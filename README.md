# できること

- 文字を無限スクロールする
- iPhoneで「ホーム画面に登録」すると全画面表示できる
- iPhoneの場合、端末横向き方向に文字が流れる
  - これが一番手間
- 設定画面
  - 文字をタップしたら設定画面
  - 背景透過
    - 流れる文字を確認しながら、変更できる
  - 流れる文字の変更
  - 文字の流れる速度を変更
  - 文字のスタイルを変更
    - 白黒反転
    - 電光掲示板風　残念ながら輝度少なし
  - 設定値はlocalStorageで保存し、次回起動時の初期値とする

chromeとiPhone-safariでしか試してません。
